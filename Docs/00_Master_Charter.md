# LocalForge Master Charter

LocalForge is a local-first, Apple-native Developer Intelligence Platform for macOS. Its core question is: what is actually true about this software, and what should happen next?

V1 is GUI-first with a thin CLI companion. Core intelligence lives in `LocalForgeCore`; the GUI and CLI call the same engines.

Default posture:

- Read-only analysis.
- Explicit folder access only.
- No telemetry by default.
- No cloud AI by default.
- No source upload by default.
- No mandatory paid runtime services.
- Unknown is never shown as verified.
