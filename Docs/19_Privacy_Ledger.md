# Privacy Ledger

Default privacy behavior:

- Local-first.
- Offline-capable.
- No telemetry by default.
- No cloud AI by default.
- No source upload by default.
- Explicit folder access only.
- Approved folders are restored through local security-scoped bookmarks when available.
- Stale, missing, or failed bookmarks are shown as access states instead of hidden failures.

Future integrations must be opt-in, revocable, disclosed, and non-essential for core functionality.
